meteor-boilerplate
==================

A meteor project structure + obvious packages for bootstrapping new meteor projects.

```
client/
  compatibility/
  views/
    admin/
    landing_pages/
      home.html
      home.js
      home.scss
    layouts
      main.html
      main.js
lib/
models/
public/
server/
```

Packages
* iron-router
* spiderable
* meteorhacks:npm
* iron-router-active
* ms-seo
* spinner
* email
* accounts-password
* accounts-ui
